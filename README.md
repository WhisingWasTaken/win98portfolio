This is a personal portfolio site styled after Windows 98.  
It recreates the classic desktop look with a taskbar, Start menu, icons, and draggable windows.
I was procrastinating this project for a while, so I am very happy this turned out the way it did.

![Preview](https://img.shields.io/badge/preview-live-success)
**Live Demo:** [whising.space](whising.space)

## Features
- Windows 98 inspired UI
- Start menu with Restart, Shutdown, and Log out
- Desktop icons that open content in app windows
- Taskbar with clock
- Custom Win98 cursor
